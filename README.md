# TodoList 管理系统

一个基于 Vue 3 的轻量级个人待办事项管理系统，支持多层级任务管理和进度跟踪。

## 功能特性

### 🚀 核心功能
- **用户管理**: 支持用户注册和登录
- **专题管理**: 创建和管理不同的工作/学习专题
- **任务管理**: 在每个专题下创建具体的任务
- **待办清单**: 为每个任务创建详细的待办事项
- **进度跟踪**: 实时显示各层级的完成进度

### 📊 进度展示
- **专题层**: 饼状图显示各专题完成情况
- **任务层**: 柱状图显示各任务完成进度  
- **清单层**: 进度条 + 备注系统，支持拖拽调整进度

### 💾 数据存储
- **智能本地文件存储**：自动在本地创建和管理JSON数据文件
- **浏览器存储**：数据存储在浏览器localStorage中（备用方案）
- 完全无感知的存储体验，无需手动管理文件
- 支持数据导出和备份
- 轻量级，无需数据库
- 数据持久化，不会因清除浏览器数据而丢失

## 技术架构

- **前端框架**: Vue 3 + Composition API
- **构建工具**: Vite
- **样式**: CSS3 + 响应式设计
- **数据存储**: 本地JSON文件 + localStorage (备用)
- **状态管理**: Vue 3 响应式系统

## 使用说明

### 数据存储模式

#### 智能本地文件存储（推荐）
- 首次使用时，系统会自动提示选择项目目录并创建数据文件
- 数据文件会自动保存在你选择的位置
- 完全无感知的存储体验，无需手动管理文件
- 数据文件可以放在项目目录下，方便管理和备份

#### 浏览器存储（备用）
- 如果浏览器不支持文件系统API，系统会自动使用localStorage
- 数据存储在浏览器中，清除浏览器数据会丢失

### 1. 用户注册/登录
- 首次使用需要注册账号
- 支持用户名和密码登录
- 数据可选择存储在本地JSON文件或浏览器中

### 2. 创建专题
- 点击"新建专题"按钮
- 输入专题名称和描述
- 专题将显示在左侧边栏

### 3. 创建任务
- 选择专题后，点击"新建任务"
- 为任务添加名称和描述
- 任务将显示在专题详情页面

### 4. 管理待办事项
- 选择任务后，点击"新建待办"
- 设置待办标题、描述、初始进度和备注
- 使用进度条调整完成进度
- 支持编辑和删除待办事项

### 5. 进度跟踪
- 实时查看各层级的完成进度
- 专题概览页面显示整体统计信息
- 支持进度条拖拽调整

## 数据模型

### 用户 (User)
```json
{
  "username": "用户名",
  "password": "密码",
  "createdAt": "创建时间"
}
```

### 专题 (Topic)
```json
{
  "id": "唯一标识",
  "name": "专题名称",
  "description": "专题描述",
  "createdAt": "创建时间",
  "tasks": ["任务数组"]
}
```

### 任务 (Task)
```json
{
  "id": "唯一标识",
  "name": "任务名称",
  "description": "任务描述",
  "createdAt": "创建时间",
  "todos": ["待办数组"]
}
```

### 待办 (Todo)
```json
{
  "id": "唯一标识",
  "title": "待办标题",
  "description": "待办描述",
  "completed": false,
  "progress": 0,
  "note": "备注信息",
  "createdAt": "创建时间"
}
```

## 自定义配置

### 修改主题色彩
在 `src/assets/main.css` 中修改 CSS 变量：
```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --success-color: #48bb78;
  --warning-color: #d69e2e;
  --error-color: #e53e3e;
}
```

### 调整进度条样式
在组件中修改进度条相关的 CSS 类：
```css
.progress-fill {
  background: linear-gradient(90deg, var(--success-color), #38a169);
}
```
